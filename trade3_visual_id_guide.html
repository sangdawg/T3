05 — Do's & Don'ts

Usage Guidelines

Rules that protect brand consistency across all applications — from product UI to pitch decks to social media.

✓ Do
  • Use Raleway 800 for hero headlines
  • Use Karla 400/500 for all body copy
  • Use Violet 500 as the primary brand colour
  • Use Orange 500 only for CTAs and accents
  • Maintain minimum 4.5:1 contrast on body text
  • Use the cream background for light-mode surfaces
  • Use Navy 500 for dark-mode base backgrounds
  • Pair Overline + Display for section headers
  • Use semantic tokens in product code
  • Test all colour usage in both modes
✗ Don't
  • Use system fonts (Arial, Inter, Roboto) in brand materials
  • Use Orange 500 as a background colour
  • Place Violet 500 text on Navy 500 backgrounds
  • Use font weights below 400 for body copy
  • Introduce new typefaces without brand approval
  • Use colours outside the defined palette
  • Use raw hex values in product code — use tokens
  • Display the logo at sizes below 24px
  • Stretch or distort the logotype
  • Use gradients as text colour without testing
